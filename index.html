Step 2

We’ll need to start with the basics of HTML understanding before we can actually set up your page, so here’s how HTML works.

HTML uses what is called a “Tag”. A tag is made up of 2 parts, a start tag and an end tag. <_> is the start tag, and </_> is the end tag. The start tag is what defines the tag, and the end tag is just the same as the start tag but with a “/” before the tag name.

HTML tags are used in like a tree. You put tags inside of other tags, such as putting paragraph tags in the “Body” tag (go to next step to understand what I'm talking about).


/* __________________________________________________________________________________

Step 3

We’ll need to actually set up our files. So, open up your “_.html” file in your text editor, and just copy and paste this code in. (Here's what it looks like in Github)

(Note, /* is used to define a comment and */ is used to end one.)


<!DOCTYPE HTML> /* This just tells it that this is in HTML and this one doesn’t need an end tag*/

<Head> /* Start tag for the “Head”, this is where you’d define the title, styles, scripts, and more */

</Head> /* This is the end of the Head tag */

<Body> /* This is the start of the “Body”, which is the tag that holds all the things we see, such as headers, paragraphs, and more

</Body> /* This is the end of the Body tag */


This is just the start very start of the tree. Every tag you use will be either in the Head or Body tag.

Step 4

We’ll now need to put text into our site. With the code we put in above, we can actually put stuff in. Put a line between the body line’s start and end tag, and then put a “<p>” tag.

This tag is used to define basic text. This tag also automatically puts a line between each instance of this tag, so you wouldn’t need to put line breaks after each one

If you do this correctly, the Body code should look like this: (In github)


<Body>

<p>

First Text

</p>

</Body>

All of these header tags have predetermined size. h1 is the biggest, while h5 is the smallest. You can use these to make things on your site.
