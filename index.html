Step 3

HTML (which is what we'll start with), makes pretty bare bones sites. The whole idea of HTML is it's structure, which is like a "tree". The coding system uses "Tags", which is anything betwen the <_> and the </_>. A tag always has a start and an end.

Your "Head" tag is everything the site runs when you open it. This would hold scripts, CSS (we'll get to this), titles, logos, and more!

Your "Body" tag is what we actually put stuff in, so like headers, paragraphs, links, and more.

Now that you've made your first tag (and message), it's time to learn new tags. There's the <h> tags (also known as the "header" tag), which is: <h1>, <h2>, <h3>, <h4>, and <h5>. All of these header tags have predetermined size. h1 is the biggest, while h5 is the smallest. You can use these to make things on your site. Each of the "Step" things on the screen are headers, along with the "Web-Seption" at the top.

With HTML being such a widespread "language", there are many other tags that you can find. I'd suggest going to www.w3schools.com/ to find more tags that you can use, like the <b> (bold) tag, <u> (underline) tag, and more!
